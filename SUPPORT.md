# Support

## Where to ask questions

- Use GitHub Discussions for setup help, usage questions, and general product conversation.
- Use GitHub Issues for reproducible bugs and concrete feature requests.

## What to include

When asking for help, include:

- ContextHUD version
- macOS version
- installation method
- exact command or workflow
- relevant logs, screenshots, or output

## Security

For sensitive security issues, use GitHub Security Advisories instead of public issues or discussions.
