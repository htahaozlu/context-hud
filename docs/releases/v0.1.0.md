# ContextHUD v0.1.0

Initial release of ContextHUD.

ContextHUD provides a local-first repository context pipeline and a native macOS HUD for Claude Code and Codex. It keeps agent-readable project briefs fresh, exposes usage visibility in a compact desktop surface, and ships with a CLI that can refresh or watch repository state continuously.

## Included in this release

- local repository context generation under `.context-hud/`
- stable outputs including `AGENT.md`, `CLAUDE.md`, `brief-now.md`, `brief-session.md`, and `brief-week.md`
- CLI commands for `hud`, `snapshot`, `watch`, `global`, and `watch-global`
- native macOS menubar companion app
- native AppKit usage window for Claude Code and Codex
- markdown and JSON artifacts for tooling and automation
- DMG packaging for macOS distribution

## Installation

1. Download `ContextHUD.dmg` from the release assets.
2. Drag `ContextHUD.app` into `Applications`.
3. Launch the app once from `Applications`.

## Notes

- The CLI remains the most dependable always-on interface.
- Repository summaries are generated locally.
- Usage aggregation depends on locally available transcript data.
