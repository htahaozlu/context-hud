# ContextBar v0.3.10

This release tightens the ContextBar rename and fixes menubar readability regressions.

## Fixed

- Dark-mode text in menubar and popover attributed strings now uses dynamic colors so labels stay readable.
- The Parallel Sessions card no longer appears unless there is a real background session to show.
- Homebrew install/upgrade guidance now points at the `htahaozlu/context-bar/context-bar` cask explicitly.
- Release tap sync now reports token permission problems clearly before the final push step.

## Install

```bash
brew update
brew upgrade --cask htahaozlu/context-bar/context-bar
```

Or download `ContextBar.dmg` from the release assets.
