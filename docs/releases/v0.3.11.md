# ContextBar v0.3.11

This release fixes Codex balance refresh parity with the official Codex usage panel.

## Fixed

- Codex 5h/7d limits now refresh from Codex's live local app-server quota endpoint instead of waiting for a new transcript `rate_limits` event.
- Codex limit text now shows remaining quota, matching the official Codex balance wording. Claude limit display is unchanged.

## Install

```bash
brew update
brew upgrade --cask htahaozlu/context-bar/context-bar
```

Or download `ContextBar.dmg` from the release assets.
